# Warehouse Management System (WMS) - System Analysis

## Overview

This project presents the analysis and design of a Warehouse Management System (WMS).  
The system focuses on logistics operations such as product storage, pallet management, order execution and workforce management.

The project applies software engineering methodologies and UML modeling techniques.

---

## System Functions

The system manages:

- Products and pallets
- Warehouse storage locations
- Driver shift management
- Customer orders
- Inventory tracking
- Temperature monitoring sensors

---

## UML Diagrams

The system design includes:

- Use Case Diagram
- Class Diagram
- Sequence Diagrams
- Component Diagram

These diagrams model system interactions and internal structure.
